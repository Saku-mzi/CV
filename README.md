# forms

A web form application with the following features:

- **User Registration Form** - Collects user information including name, email, and password
- **Password Requirements** - Enforces minimum 8 characters with uppercase, lowercase, numbers, and symbols
- **Radio Buttons** - Food preference selection (chicken, beef, fish)
- **Text Area** - Bio/description field
- **Dropdown Menu** - South African province selector with 8 provinces
- **Checkbox** - Terms of service agreement
- **Submit Button** - Routes to thank-you.html on form submission

**Styling:**

- Monospace font with light green background
- Rounded input borders with box shadow on button
- Centered layout with form styling
- Responsive design considerations
